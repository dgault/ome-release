  • OMERO.mtools is a suite of tools written in MATLAB for performing common analysis tasks on images stored in an OMERO server. Please see the Product page for information.

  • Please note that this version of OMERO.mtools is compatible with OMERO 5.3 servers. If you require compatibility with an OMERO 5.2 server or earlier, previous versions are available from the main downloads folder.

  • Choose the version of OMERO.mtools for your platform (Windows/Mac) below.

  • The installer will fetch the 2016b MCR from Mathworks and install it for you if you do not already have it, but it may be more efficient to install it manually from the table below before installing OMERO.mtools.

  • Once downloaded simply unzip and run. Running from a network drive is not recommended.

  • Source code is available on GitHub.
